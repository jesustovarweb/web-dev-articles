# WordPress

Este directorio contiene ejemplos de código y snippets relacionados con WordPress, extraídos de artículos publicados en mi web y basados en situaciones reales de desarrollo.

El objetivo no es cubrir todos los aspectos de WordPress, sino documentar soluciones concretas que siguen siendo válidas y mantenibles en proyectos profesionales actuales.

## Organización

Los ejemplos se agrupan por contexto o plugin cuando tiene sentido hacerlo:

- `core/`  
  Snippets y ajustes que afectan al comportamiento general de WordPress, sin depender de plugins concretos.

- Directorios por plugin (por ejemplo, `woocommerce/`, `wpml/`, etc.)  
  Personalizaciones específicas relacionadas con plugins habituales en proyectos reales.

Cada ejemplo vive en su propio directorio e incluye su propio README con una explicación detallada y un enlace al artículo original del que proviene.

## Criterio de selección

No todo el código que aparece en los artículos del blog se publica aquí de forma automática.

Solo se incluyen ejemplos que:

- Representan una forma profesional de trabajar.
- Pueden reutilizarse con seguridad en otros proyectos.
- Tienen sentido fuera del contexto completo del artículo.

El resto del código se mantiene únicamente como parte del contenido editorial.

## Notas

El código suele estar documentado en inglés, siguiendo una convención habitual en proyectos de desarrollo, aunque los artículos y la documentación principal del repositorio estén en español.
