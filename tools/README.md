# Herramientas

Este directorio contiene herramientas y utilidades auxiliares relacionadas con el desarrollo web, creadas o adaptadas para resolver necesidades concretas en proyectos reales.

A diferencia de otros directorios del repositorio, aquí no se incluyen snippets de código pensados para integrarse directamente en un proyecto, sino herramientas puntuales que sirven de apoyo durante el desarrollo, mantenimiento o análisis técnico.

## Qué tipo de contenido hay aquí

En este directorio pueden encontrarse, por ejemplo:

- Scripts de uso puntual o herramientas de línea de comandos
- Utilidades para depuración, migraciones o tareas repetitivas
- Herramientas internas creadas para resolver un problema concreto
- Pequeños proyectos o helpers que no encajan en frontend, backend o WordPress

El enfoque es siempre práctico y orientado a problemas reales, no a crear herramientas genéricas sin un caso de uso claro.

## Criterio de inclusión

Solo se incluyen herramientas que:

- Han sido utilizadas en proyectos reales.
- Aportan valor como referencia técnica.
- Tienen sentido fuera del contexto exacto en el que se crearon.

Herramientas demasiado específicas o difíciles de reutilizar se mantienen fuera del repositorio.

## Documentación

Cada herramienta vive en su propio directorio e incluye un README con una explicación clara de su propósito, uso y contexto, así como un enlace al artículo del blog del que proviene, cuando aplica.
